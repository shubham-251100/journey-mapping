# SarasAI

A Journey Tracking application built with Vite.

## Features

### Adding a New Card

To add a new card to your journey:

1. Click on the Logo icon
2. A new card will be added to your list

### Updating Progress

To update the progress:

1. Click on any card in your list
2. The progress will be updated automatically

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- Vite
- JavaScript
